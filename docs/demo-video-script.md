# Demo Video Script

Target length: 75 to 90 seconds.

## 0:00 to 0:15 - Problem

Show the dashboard header and seller machine.

Voiceover:

Autonomous machines and AI agents increasingly need to buy tiny units of real-world resources, such as electricity, compute, data, and network access. Card rails and normal crypto transfers are too heavy for high-frequency machine decisions.

## 0:15 to 0:35 - Solution

Show Solar Node A17 and the changing spot price.

Voiceover:

MachinePay Grid creates a stablecoin-native resource market. A seller machine publishes price and capacity. A buyer agent watches the signal and decides whether the next 10 seconds of power is profitable.

## 0:35 to 0:55 - Payment Loop

Click Force Buy once, then let the simulation run.

Voiceover:

When margin is positive and the hourly spend policy allows it, the buyer creates a USDC nanopayment authorization. The ledger shows amount, machine power, route, and authorization status.

## 0:55 to 1:10 - Arc and Circle Proof

Zoom or point at the proof packet and architecture row.

Voiceover:

Circle provides the USDC payment rail through Nanopayments or Gateway. Arc becomes the settlement layer where batched machine payments can be finalized and proven.

## 1:10 to 1:25 - Roadmap

Show the README or proposal file briefly.

Voiceover:

The current demo is a pitch-ready MVP. The next implementation step is replacing mocked authorizations with Circle test wallets, adding real Arc Testnet transaction proofs, and expanding the marketplace to compute and API data.

## Call to action

Voiceover:

MachinePay Grid turns Arc and Circle into the payment fabric for autonomous machine commerce.

## Recording checklist

- Start from a reset state.
- Click Force Buy four times so the Arc batch proof appears.
- Keep the proof packet visible for at least 5 seconds.
- Show the README for one short moment at the end.
- Export as 1080p MP4 if possible.
